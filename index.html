📌 行程总览

路线：桂林市区（象鼻山/两江四湖/芦笛岩）→ 阳朔（遇龙河/十里画廊/银子岩/西街/兴坪渔村）→ 龙脊梯田（平安/金坑大寨）。城市间以高铁/大巴为主，县内以网约车/骑行/竹筏为主。

核心交通建议

到达：桂林两江国际机场 / 桂林站 / 桂林北站
市区 ⇄ 阳朔
高铁 28–40 分钟（桂林/桂林北 → 阳朔站），或大巴约 1.5h；阳朔站需转车 25–35 分钟至县城/西街。
阳朔 ⇄ 兴坪
县城汽车站/网约车 40–60 分钟；骑行去十里画廊/遇龙河更灵活。
桂林 ⇄ 龙脊
桂林汽车站直达平安/大寨约 2–3 小时（山路弯多，晕车带药）。

住宿建议

建议2+3+2晚（市区/阳朔/龙脊）
桂林市区 2晚
两江四湖/中心广场附近步行可达景点，夜游方便。

龙脊 2晚
平安/大寨山顶观景房，日落/日出步行可达（带轻便行李上山）。

D1｜抵达桂林 · 两江四湖夜游

轻松
动线
抵达 → 酒店放行李 → 正阳步行街/中心广场 → 两江四湖夜游（可选船票）
看点
日月双塔夜景、榕杉湖灯光、水上古桥群、城市夜色
餐饮
米粉初体验（晚到也能吃）｜夜宵：啤酒鱼（清淡版更适合首日）
交通
机场大巴/地铁/网约车至市中心；夜游码头步行可达
预算
餐饮 80–120｜夜游船 160–220（可视淡旺季）
✅ 船票提前预定可选好时间段；如遇下雨，夜游照常且更有倒影氛围。
⚠️ 夜游散团点与上船点可能不同，留意导航与打车位置。

D2｜象鼻山·芦笛岩·老码头漫步

城市慢游
动线
象鼻山公园 → 伏波山/叠彩山（二选一）→ 芦笛岩 → 杉湖/老码头傍晚散步
看点
象鼻山打卡、溶洞奇观灯光秀、桂林城景远眺
餐饮
午：田螺酿/漓江虾｜晚：椿记烧鹅/地方家常菜
交通
市内公交/网约车最省心；芦笛岩离市区 6–7km
预算
门票合计 150–220（视联票）｜餐饮 100–150
✅ 溶洞拍照建议用广角 + 手电补光，避开强对比光位。
⚠️ 象鼻山内外拍点众多，若不入园也可在滨江路远景构图。

D3｜桂林 → 阳朔 · 西街夜生活

转场 + 探索
动线
高铁/大巴前往阳朔 → 入住（西街或十里画廊一带）→ 骑行熟悉路线 → 西街夜市/酒吧
看点
喀斯特群峰、乡村田园小路、江畔黄昏
餐饮
午：阳朔啤酒鱼/柚子皮/芋头扣肉｜晚：西街各国小馆
交通
阳朔站→县城需转接驳；或桂林直达大巴至阳朔汽车站
预算
交通 30–90（视方式）｜餐饮 120–180
✅ 入住后先熟悉西街至十里画廊骑行动线，第二天更顺手。
⚠️ 西街夜间人多，财物与酒精饮品安全注意。

D4｜遇龙河竹筏 · 十里画廊骑行 · 银子岩

经典阳朔
动线
早竹筏（遇龙河上/下游二选一）→ 十里画廊骑行（图腾古道/大榕树/月亮山沿线）→ 下午银子岩溶洞
看点
清澈河面、倒影山峦、田园人家、洞内石幔奇景
餐饮
午：沿线农家菜｜晚：返回县城小吃/侗族酸汤火锅
交通
竹筏需预约/现场排队；骑行租电助力更轻松；银子岩在县区外需打车/小巴
预算
竹筏 140–260（段线不同）｜银子岩 70–90｜餐饮 120
✅ 竹筏更推荐清晨或傍晚顺光时段；带防水袋与轻薄雨衣。
⚠️ 高温季注意补水与防晒；骑行尽量走机耕小路更安全有景。

D5｜兴坪古镇 · 漓江（九马画山/黄布倒影）· 相公山日落

山水大片
动线
早去兴坪古镇/渔村散步 → 漓江徒步或电动竹筏（合法航段）→ 傍晚登相公山拍日落（或次日清晨看日出）
看点
二十元背景山水、渔火人家、相公山峰丛云海
餐饮
午：兴坪粉店/江边小馆｜晚：返回阳朔或住兴坪（方便清晨机位）
交通
阳朔—兴坪车程 40–60 分钟；相公山山道规范停车后需徒步/接驳
预算
兴坪/相公山门票及接驳 60–120｜餐饮 120
✅ 相公山更看天气与云量；若阴雨，可改银子岩/蝴蝶泉等室内点。
⚠️ 漓江航段需遵守当地管理，避免上非正规竹筏。

D6｜阳朔 → 龙脊梯田（平安/大寨）· 日落

转场 + 轻徒步
动线
早退房 → 车赴龙胜 → 入住山里观景民宿 → 下午轻徒步至观景点看日落
看点
层层梯田、苗侗风情、云雾流光
餐饮
午：途中服务区简单补给｜晚：山里长桌宴/竹筒饭
交通
阳朔/桂林出发班车或包车 2–3 小时；景区内观光车上山后再步行
预算
门票+观光车 120–150｜餐饮 120｜住山顶行李托运 30–50
✅ 只带轻便背包上山，把大件行李寄存在山脚更省力。
⚠️ 山路弯多容易晕车，提前吃晕车药；雨后步道防滑。

D7｜龙脊日出 · 返回桂林/出发地

收官
动线
清晨看日出 → 回民宿早餐 → 下山返桂林 → 返程（火车/飞机）
看点
金坑/平安观景台云海线条，天气好能拍到光影层次
餐饮
当地腊味/土鸡汤｜返程路上简单补给
交通
景区车下山接驳 → 桂林站/机场
预算
返程交通视城市而定；桂林—机场打车 40–60 分钟
✅ 若返程较晚，可在桂林市区再补拍日景/买手信。
⚠️ 山区天气多变，给相机/手机准备干燥包与防潮。

🍜 必吃美食清单

桂林市区

  • 桂林米粉（卤蛋/脆皮/酸笋按喜好）
  • 田螺酿｜椿记烧鹅｜漓江虾
  • 马蹄糕/荔浦芋制品

阳朔 & 龙脊

  • 啤酒鱼（清淡少辣版更通用）
  • 柚子皮/芋头扣肉/侗族酸汤火锅
  • 竹筒饭/长桌宴/糍粑

小贴士：米粉讲究“卤水味”，第一次可点“全配”，再按喜好调整配菜与酸辣度。
